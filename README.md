# Brazil-E-Commerce-Sales-Analytics-Dashboard
🛒 Brazilian E-Commerce Sales Analytics Dashboard

📌 Project Overview

This project is an end-to-end Data Analytics and Business Intelligence solution built using the Brazilian E-Commerce Public Dataset by Olist. The objective was to transform raw transactional data into meaningful business insights using SQL Server (SSMS), SQL, Python, Power BI, Power Query, and DAX.

The project covers the complete analytics workflow, including data cleaning, database creation, SQL analysis, data modeling, KPI development, and interactive dashboard creation.

📂 Dataset

Dataset: Brazilian E-Commerce Public Dataset by Olist

🔗 Kaggle Dataset: https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

The dataset contains information about:

* Customers

* Orders

* Order Items

* Products

* Sellers

* Payments

* Reviews

* Product Categories

* Geolocation

📋 Project Workflow

1️⃣ Data Preparation

* Imported multiple CSV files from the Olist dataset.

* Cleaned missing values and duplicate records using Python (Pandas).

* Performed additional transformations using Power Query.

* Standardized data types and prepared data for reporting.

2️⃣ Database Management (SQL Server)

* Imported datasets into SQL Server (SSMS).

* Created relational database tables.

* Managed product, customer, seller, payment, and order data.

* Used SQL Server to perform business analysis.

3️⃣ SQL Analysis

Performed analysis using:

* INNER JOIN

* LEFT JOIN

* GROUP BY

* HAVING

* Aggregate Functions

* CTEs (Common Table Expressions)

* Window Functions

* Ranking Functions

* Views
